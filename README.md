Proyecto Análisis de Ventas BData Analytics

Proyecto de Data Analytics desarrollado para analizar el rendimiento de ventas mediante un flujo completo de analítica de datos, desde la limpieza y transformación de la información hasta la construcción de un dashboard ejecutivo para la toma de decisiones.

Microsoft Excel
Power Query
SQL Server

Modelo Estrella (Star Schema)
DAX
Microsoft Power BI

Objetivos
Limpiar y estandarizar datos provenientes de diferentes fuentes.
Diseñar un modelo dimensional para optimizar el análisis.
Obtener indicadores clave mediante consultas SQL.
Construir medidas DAX para análisis dinámicos.
Desarrollar un dashboard interactivo que facilite la toma de decisiones.
